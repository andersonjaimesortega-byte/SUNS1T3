# Mobile Brand Implementation Skill

Logic for applying corporate identity to mobile-first components in the SunSite PWA.

## Components Covered
- **Form Selectors**: Using corporate green for selection highlights.
- **Voice Module**: Branded mic animations and status indicators.
- **Resources Module**: Branded inputs for Personnel and Materials.

## Technical Requirements
- POppins font loading via Google Fonts or Local assets.
- CSS Variable injection in Vite config or Index.css.
